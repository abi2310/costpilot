# CostPilot — Schätzung technischer Fertigungskosten

Ein leichtgewichtiges Projekt zur Modellierung und Vorhersage von Fertigungskosten anhand technischer Bauteil‑ und Prozessmerkmale. Dieses Repository enthält Datensätze, exploratory notebooks, simple Model‑Pipelines (lineare Modelle, Decision Trees, Boosting) sowie Benchmark‑Ergebnisse. 

## Kurzbeschreibung

Kosten für Bauteile (z. B. Herstellungspreise) werden aus physikalischen und prozessbezogenen Input‑Variablen vorhergesagt: Gewicht, Fläche, Materialstärke, Materialtyp, Anzahl von Bearbeitungsschritten etc.

Das Projekt zeigt, wie: 
- Daten eingelesen und bereinigt werden
- einfache, interpretable lineare Modelle trainiert und evaluiert werden
- Tree‑basierte Modelle (Decision Tree, CatBoost) getestet und verglichen werden
- Modellmetriken dokumentiert und Feature‑Importances visualisiert werden

Warum das Projekt: Recruiter/Domain‑Reviewer erhalten so Einblick in Data‑Science‑Praxis (E2E): Datenaufbereitung, Modellwahl, Evaluation und Ergebnisinterpretation.
